	🚧 This project is actively under development

	Current Features 🌟

	Send food photos through WhatsApp for instant analysis
	Get nutritional information (calories, protein, carbs, fat)
	Track daily nutritional intake
	Set and monitor personal nutrition goals
	Full Hebrew language support
	No app installation required - works through WhatsApp

	Tech Stack 💻

	Python/Flask backend
	GPT-4 Vision for food analysis
	Twilio WhatsApp API
	SQLite database (PostgreSQL migration in progress)
